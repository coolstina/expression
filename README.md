# expression

Go regular expression wrapper.

## Installation

```shell script
go get -u github.com/coolstina/expression
```


## Features

- Match IPv4
- Match internal IPv4
- Match cellphone Of China
- Match telephone Of China
- Match email address
- Match web domain
- Match time hour point
- Match string
